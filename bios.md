# entering bios

press F1, F2, Del simultaneously
lenovo -> Fn+F2
del -> F12

# bios settings

* check the video

- time/clock: month/day/year
- ahci
- uefi
- other settings are preffered to be fast:
  - fast boot
  - gpu graphics
- graphic 128M

# flashing bios:

- battery must exist in laptop ( +20% )
- observe for possible errors

failed? -> resolve the issue and try again (e.g. might need a ram change or cleanup)
