# Apache maven

is a project management tool that uses a POM and provides a standardized project structure.

a POM is an XML file that contains information about the project and configuration details
(e.g configuration and dependencies).

[maven directory layout](100/README.md)
[maven dependency managment](102/README.md)

Apache Maven *WAR Plugin* builds a Web Application Archive (WAR) file from the project output
and its dependencies.
