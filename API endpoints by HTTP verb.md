
refers to organizing or documenting the available API routes (endpoints) based on the HTTP method they use — like GET, POST, PUT, DELETE, etc.

[[API endpoints]]

GET:

    GET /users – get all users

    GET /users/:id – get user by ID

POST:

    POST /users – create a new user

PUT:

    PUT /users/:id – update a user completely

PATCH:

    PATCH /users/:id – update part of a user’s data

DELETE:

    DELETE /users/:id – delete a user
