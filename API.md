# API

An *API* (Application Programming Interface) is a set of rules and protocols that allows
different software applications to communicate with each other. In essence, an API defines
how a client (like a web browser or mobile app) can request data or services from a server,
and how that server should respond.

# REST API

A REST API (Representational State Transfer API) is a specific type of API that follows a
set of architectural principles known as REST. REST was designed to make web services easier
to use and integrate. It is widely used for APIs because it's simple, flexible, and works
well with the HTTP protocol, which is the foundation of the web.

A REST API allows applications to communicate over HTTP by defining a set of rules for how
to format requests and responses.
